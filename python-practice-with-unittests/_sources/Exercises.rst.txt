Exercise Overview
=================

Exercise 101
------------

Ask user for name and print 'Hello, <name>'

Exercise 102
------------

Ask user which number in the Fibonacci sequence they would like and print it

Exercise 201
------------

Covert between miles/hour and kilometers/hour

Exercise 301
------------

Extract certain text from a file. For example year, email address and words
beginning with a capital letter.

Exercise 302
------------

Extract different parts of string and put them into a dictionary.
This exercises uses tyre size codes.

Exercise 401
------------

This has all the code but there is a bug causing a test to fail. Can you
find the bug?
